## Методи встановлення

### 1. WSL (Windows Subsystem for Linux)

Найпростіший спосіб — без створення окремого розділу диска.

**Вимоги:** Windows 10 версія 2004+ або Windows 11.

**Встановлення:**

```powershell
# Відкрити PowerShell від імені адміністратора
wsl --install
```

За замовчуванням встановлюється Ubuntu. Щоб вибрати дистрибутив:

```powershell
wsl --list --online          # переглянути доступні дистрибутиви
wsl --install -d Debian      # встановити конкретний дистрибутив
```

Після встановлення перезавантажити ПК. При першому запуску створити ім'я користувача та пароль.

---

### 2. Dual Boot (подвійне завантаження)

Повноцінна інсталяція Linux поруч з Windows. Linux отримує окремий розділ диска.

**Кроки:**

1. **Завантажити ISO-образ** дистрибутива (наприклад, Ubuntu).

2. **Створити завантажувальну флешку** (мінімум 8 ГБ):
   - Завантажити Rufus (Windows)
   - Вибрати ISO → схема розділів GPT → старт

3. **Звільнити місце на диску** в Windows:
   - `Win + X` → Керування дисками → ПКМ на розділі → Стиснути том
   - Рекомендований розмір: 30–50 ГБ

4. **Вимкнути Fast Startup у Windows:**
   - Панель керування → Електроживлення → Дії кнопок живлення → Вимкнути швидке завантаження

5. **Завантажитися з флешки:**
   - Перезавантажити ПК → натиснути `F2`/`F12`/`Del` (залежить від виробника) → вибрати USB у Boot Menu

6. **Встановити Linux:**
   - Вибрати "Install Ubuntu alongside Windows"
   - Розподілити місце між системами
   - Вказати ім'я користувача, пароль, часовий пояс

7. **Після встановлення** з'явиться меню GRUB — вибір ОС при кожному завантаженні.

---

### 3. Встановлення у Virtual Box (віртуальна машина)

Linux запускається всередині Windows як програма. Не потребує зміни розділів диска.

**Кроки:**

1. Завантажити та встановити VirtualBox.
2. Завантажити ISO-образ Linux.
3. У VirtualBox: `Новий` → вказати назву, тип Linux, версію.
4. Виділити RAM (рекомендовано 2–4 ГБ) та місце на диску (20+ ГБ).
5. У налаштуваннях VM: `Носії` → підключити ISO до оптичного приводу.
6. Запустити VM → пройти стандартну інсталяцію Linux.
