## Навігація файловою системою

### `ls` — перелік файлів і каталогів

Виводить вміст поточного або вказаного каталогу.

```bash
ls                        # вміст поточного каталогу
ls /home/user/Documents   # вміст вказаного каталогу
```

### `ls -R` — рекурсивний перелік

Виводить вміст каталогу та всіх вкладених підкаталогів.

```bash
ls -R /var/log
```

### `ls -a` — показати приховані файли

Файли, що починаються з `.`, є прихованими. Прапор `-a` (all) їх відображає.

```bash
ls -a ~
```

### `ls -al` — детальний список включно з прихованими

Комбінує `-a` та `-l`: відображає права доступу, власника, розмір, дату.

```bash
ls -al /etc
```

### `cd` або `cd ~` — перехід до домашнього каталогу

```bash
cd
cd ~
```

### `cd ..` — перехід на рівень вище

```bash
cd ..              # з /home/user/docs → /home/user
```

### `cd` (без аргументів) — повернення до home

```bash
cd                 # еквівалент cd ~
```

### `cd /` — перехід до кореневого каталогу

```bash
cd /
ls                 # bin  boot  dev  etc  home  lib  ...
```

## Робота з файлами

### `cat > filename` — створення файлу з введенням з клавіатури

Перенаправляє стандартний ввід у файл. Завершення: `Ctrl+D`.

```bash
cat > notes.txt
Це перший рядок.
Це другий рядок.
^D
```

### `cat filename` — виведення вмісту файлу

```bash
cat /etc/hostname
cat readme.txt
```

### `cat file1 file2 > file3` — об'єднання файлів

Зливає `file1` та `file2` в новий файл `file3`.

```bash
cat header.txt body.txt > full_document.txt
```

---

### `mv file "new file path"` — переміщення файлу

```bash
mv report.txt /home/user/Documents/report.txt
mv "my file.txt" /tmp/"my file.txt"
```

### `mv filename new_file_name` — перейменування файлу

```bash
mv old_name.txt new_name.txt
```

---

### `rm filename` — видалення файлу

```bash
rm temp.log
rm -r old_folder/     # рекурсивне видалення каталогу
rm -f file.txt        # примусове видалення без підтвердження
```

> **Увага:** `rm` не переміщує в кошик — файл видаляється без можливості відновлення.

## Робота з каталогами

### `mkdir directoryname` — створення каталогу

```bash
mkdir projects
mkdir -p projects/2024/january    # створення вкладених каталогів
```

### `rmdir` — видалення порожнього каталогу

```bash
rmdir empty_folder
```

Якщо каталог не порожній — використовувати `rm -r`.

### `mv` — переміщення або перейменування каталогу

```bash
mv old_dir/ new_dir/
mv folder/ /home/user/backup/folder/
```

## Системні команди

### `sudo` — виконання команди з правами суперкористувача

```bash
sudo apt update
sudo rm /etc/protected_file
sudo nano /etc/hosts
```

### `man` — довідкова сторінка команди

```bash
man ls
man grep
man sudo
```

Навігація: `Space` — наступна сторінка, `q` — вихід.

### `history` — журнал введених команд

```bash
history
history | grep apt         # пошук команд з apt
!42                        # виконати команду №42 з history
```

### `clear` — очищення терміналу

```bash
clear
# або Ctrl+L
```

### `uname` — інформація про ядро та систему

```bash
uname -a      # всі дані: ядро, хост, версія, архітектура
uname -r      # лише версія ядра
uname -s      # назва ОС (Linux)
```

### `lsb_release` — інформація про дистрибутив

```bash
lsb_release -a          # повна інформація про дистрибутив
lsb_release -d          # лише опис (наприклад, Ubuntu 24.04 LTS)
```

## Команди `pr` — форматування тексту для друку

### `pr -x` — розбивка на стовпці

```bash
pr -3 file.txt            # вивести у 3 стовпці
```

### `pr -h` — встановлення заголовка

```bash
pr -h "Мій звіт" file.txt
```

### `pr -n` — нумерація рядків

```bash
pr -n file.txt
```

## Команда `cut` — вирізання полів з тексту

Витягує частини рядків за роздільником або позицією символів.

```bash
cut -d':' -f1 /etc/passwd       # перше поле (логіни користувачів)
cut -d',' -f2,4 data.csv        # поля 2 і 4 з CSV
cut -c1-10 file.txt             # перші 10 символів кожного рядка
```
